## 📌 О проекте

**Умные Заметки** — это десктопное приложение с графическим интерфейсом для управления личными заметками.
Приложение позволяет создавать, редактировать и удалять заметки, присваивать им теги
и осуществлять фильтрацию по тегам. Все данные сохраняются локально в `JSON`-файл
и автоматически загружаются при следующем запуске.

---

## ✨ Функциональность

### 📄 Управление заметками
- ➕ **Создание** новой заметки через диалоговое окно с вводом названия
- 💾 **Сохранение** текста и тегов текущей заметки
- 🗑️ **Удаление** выбранной заметки из списка и хранилища
- 👁️ **Просмотр** заметки по клику в списке

### 🏷️ Управление тегами
- ➕ **Добавление тега** к выбранной заметке (без дублирования)
- ❌ **Удаление тега** из выбранной заметки
- 🔍 **Поиск заметок по тегу** — фильтрует список заметок в реальном времени
- 🔄 **Сброс поиска** — возвращает полный список заметок

### 💾 Хранение данных
- Автоматическое **сохранение в JSON** при каждом изменении
- Автоматическая **загрузка данных** при запуске приложения
- Стартовая заметка-инструкция при первом запуске

---

## 🗃️ Формат хранения данных

```json
{
    "Название заметки": {
        "текст": "Содержимое заметки...",
        "теги": ["тег1", "тег2"]
    }
}
Данные сохраняются в файл notes_data.json в директории запуска приложения.

🛠️ Технологии
Технология	Применение
Python 3.12	Язык разработки
PyQt5	Графический интерфейс (GUI)
QTextEdit	Многострочное поле редактирования текста заметки
QListWidget	Списки заметок и тегов с поддержкой выбора
QLineEdit	Поле ввода нового тега
QPushButton	Кнопки управления заметками и тегами
QInputDialog	Диалоговое окно для ввода названия заметки
QVBoxLayout / QHBoxLayout	Компоновка виджетов
json	Сериализация и десериализация данных
os	Проверка существования файла, работа с путями
🚀 Запуск
1. Клонируй репозиторий
Bash

git clone https://github.com/dibofailet-hash/clever_notebook
cd smart-notes
2. Установи зависимости
Bash

pip install PyQt5
3. Запусти приложение
Bash

python main.py
⚠️ Важно: При первом запуске автоматически создаётся файл notes_data.json с демонстрационной заметкой-инструкцией.
